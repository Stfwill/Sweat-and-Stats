# Sweat & Stats

**Every Rep Counts.**

Version 2.2 completes key portions of the Exercise Library and Workout workflow and adds Exercise Library import/export.

## Included in v2.2

- Equipment Type filter in the Exercise Library
- `Other` as an Equipment Type
- Custom Equipment Names saved for reuse
- Post-workout summary screen
- Required workout mood before saving
- Overall workout notes
- Workout totals for duration, exercises, sets, and estimated volume
- Import one or multiple exercises from the standard text format
- Validation and smart matching for common equipment aliases
- Duplicate handling: Skip, Replace Existing, or Create Copy
- Import preview and results summary
- Copy Exercise text in the same standardized format
- Centralized master data for muscle groups, movement patterns, and equipment types
